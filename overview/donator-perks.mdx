## How to obtain Donator Perks

Donator perks can be purchased via **server subscriptions** in our [Support Server](https://adore.rest/invite) which can be found at the top of the channel list labeled as **Server Shop**.

</Frame>
<Warning>
  We accept payments via **PayPal** and **cryptocurrency**, including Bitcoin (BTC), 
  Litecoin (LTC), and Ethereum (ETH).
</Warning>

    <AccordionGroup>
    <Accordion title="ChatGPT">
      Ask ChatGPT a question.
      <CodeGroup>
      ```javascript Syntax
      ,chatgpt (question)
      ```
      ```javascript Example
      ,chatgpt am i cute?
      ```
      </CodeGroup>
    </Accordion>
    <Accordion title="Self Alias">
      Create personal command aliases that work across all servers.
      <CodeGroup>
        ```javascript Syntax
        ,alias self add (alias) (command)
        ```
        ```javascript Example
        ,alias self add ban ,ban {user} {reason}
        ```
      </CodeGroup>
    </Accordion>
    <Accordion title="Self Prefix">
      Set your own custom command prefix across all servers that utilize adore.
      <CodeGroup>
        ```javascript Syntax
        ,prefix self (prefix)
        ```
        ```javascript Example
        ,prefix self j
        ```
      </CodeGroup>
    </Accordion>
    <Accordion title="Background Removal">
      Remove the background from an image.
      <CodeGroup>
        ```javascript Syntax
        ,transparent (image URL)
        ```
        ```javascript Example
        ,transparent https://adore.bot/epic_face.png
        ```
      </CodeGroup>
    </Accordion>
    <Accordion title="Last.fm Mode">
      Customize the embed layout of the `nowplaying` command.
      <CodeGroup>
        ```javascript Syntax
        ,lastfm mode (embed script)
        ,lastfm mode (check|reset)
        ```
        ```javascript Example
        ,lastfm mode {embed}$v{title:Now Playing}$v{description: {track.name}}$v{thumbnail: {album.cover}}
        ```
      </CodeGroup>
    </Accordion>
    </AccordionGroup>
  </Tab>
</Tabs>
